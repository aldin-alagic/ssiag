"ICEBEIN CHILLER I"

Der "icebein chiller I" kühlt das Wasser innerhalb von 10 Minuten von 20° Celsius auf bis auf eine Minimal-Temperatur von 5° Celsius herab. Dabei lässt sich die gewünschte Temperatur auch individuell am Display des Chillers einstellen. Auf Knopfdruck wird das gekühlte Wasser kontinuierlich durch die "icebein legs I" gepumpt. So wird eine konstante Kühlung der Muskulatur gewährleistet. Einfach Wasser* in den Chiller füllen, kühlen und regenerieren. *Bitte beachten Sie, dass wir ausdrücklich die Verwendung von destilliertem Wasser unter Zugabe von "Oxylythe" empfehlen! Dadurch halten Sie das "icebein recovery system" sauber, frei von Keimen und Bakterien und beugen Sie der Bildung von Schimmel im System vor. Desinfektionsmittel "Oxylythe" ist seperat im erhältlich.
Mase Chiller: ca. 448 x 330 x 440mm
Netzbetrieb Voltage: 220 ~ 240 Volt
Frequenz: 50 Hz
